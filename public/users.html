Tasks API End points


Check the Available API Endpoints for Tasks at

  • Create Tasks (POST Request):

    https://pern-21um.onrender.com/api/tasks


    Make Sure you create a user before this Send JSON in the body of the request with {"name" : "taskname","userId": "id of the created user"} also make sure note the created id for testing with other API endpoints

  • See All Tasks cretaed by all users (GET Request) :

    https://pern-21um.onrender.com/api/tasks

  • See All Tasks cretaed by specific user (GET Request) :

    https://pern-21um.onrender.com/api/users/tasks

    Make Sure you create a user before this Send JSON in the body of the request with {"userId": "id of the created user"}

  • Update Task by Task ID (PUT Request) :

    https://pern-21um.onrender.com/api/tasks/{task_id}

    Make Sure you create a task before this Send JSON in the body of the request with {"status": "COMPLETED|IN_PROGRESS|FAILED"} use one of the options completed or in_progress or failed don't put more than one put your created task id in the place of {task_id}


  • Delete Task by Task ID (PUT Request) :

    https://pern-21um.onrender.com/api/users/tasks/{task_id}

    Make Sure you create a task before this Send JSON in the body of the request with {"status": "COMPLETED|IN_PROGRESS|FAILED"} use one of the options completed or in_progress or failed don't put more than one example {"status": "COMPLETED"} put your created task id in the place of {task_id}
