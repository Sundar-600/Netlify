User API End points


Check the Available API Endpoints for User at

  • Create Users (POST Request):

    https://pern-21um.onrender.com/api/users


    Send JSON in the body of the request with {"name" : "username"}, Also note the created id of the created user for testing with other Endpoints where it's needed

  • See All Users (GET Request) :

    https://pern-21um.onrender.com/api/users

    Also if you include ?name query parameter it will filter the user by name Example : https://pern-21um.onrender.com/api/users?name="username"

  • See Specific User (GET Request) :

    https://pern-21um.onrender.com/api/users/{id}


    Send the id of your user or known user on that {id} place holder

  • Update Specific User (PUT Request) :

    https://pern-21um.onrender.com/api/users/{id}


    Send the id of your user or known user on that {id} place holder and Send JSON in the body of the request with {"name":"new_username"}

  • Delete Specific User (DELETE Request) :

    https://pern-21um.onrender.com/api/users/{id}


    Send the id of your user or known user on that {id} place holder to delete Note : It will delete all the task created by the particular user as well
